Sección 1: Estructuras de Control

Bucle while

Definición: Se utiliza cuando no sabemos cuántas veces se repetirá un bloque de código, pero sí conocemos la condición que debe cumplirse.

Sintaxis:

while (condición) {
    // Código a ejecutar
}

Ejemplo: Contar del 1 al 10

int i = 1;
while (i <= 10) {
    System.out.println(i);
    i++;
}

Bucle do-while

Definición: Similar al while, pero garantiza que el código se ejecutará al menos una vez.

Sintaxis:

do {
    // Código a ejecutar
} while (condición);

Ejemplo: Menú simple

int opcion;
do {
    System.out.println("1. Opción A");
    System.out.println("2. Opción B");
    System.out.println("3. Salir");
    opcion = scanner.nextInt();
} while (opcion != 3);

Bucle for

Definición: Ideal para recorrer un número conocido de repeticiones.

Sintaxis:

for (inicialización; condición; incremento) {
    // Código a ejecutar
}

Ejemplo: Recorrer un arreglo

int[] numeros = {1, 2, 3, 4, 5};
for (int j = 0; j < numeros.length; j++) {
    System.out.println(numeros[j]);
}

Sección 2: Clases, Objetos y Métodos

Clases y Objetos

Definición: Una clase es como un molde, y los objetos son las instancias creadas a partir de ese molde.

class Coche {
    String color;
    String marca;
    int velocidad;
}

public class Main {
    public static void main(String[] args) {
        Coche coche1 = new Coche();
        coche1.color = "Rojo";
        coche1.marca = "Toyota";
        coche1.velocidad = 120;

        Coche coche2 = new Coche();
        coche2.color = "Azul";
        coche2.marca = "Ford";
        coche2.velocidad = 100;
    }
}

Métodos

Definición: Son comportamientos que los objetos pueden realizar.

class Coche {
    void acelerar() {
        System.out.println("El coche está acelerando");
    }
}

Sobrecarga de Métodos

class Calculadora {
    int sumar(int a, int b) {
        return a + b;
    }

    double sumar(double a, double b) {
        return a + b;
    }
}

Modificador static

class Utilidad {
    static int contadorDeInstancias = 0;

    static void mostrarMensaje() {
        System.out.println("Este es un método estático.");
    }
}

Clases Locales

class Main {
    void metodoConClaseLocal() {
        class ClaseLocal {
            void mostrar() {
                System.out.println("Soy una clase local.");
            }
        }
        ClaseLocal obj = new ClaseLocal();
        obj.mostrar();
    }
}

Sección 3: Pilares y Conceptos Avanzados

Encapsulamiento

class CuentaBancaria {
    private double saldo;

    public void depositar(double cantidad) {
        saldo += cantidad;
    }

    public void retirar(double cantidad) {
        if (saldo >= cantidad) {
            saldo -= cantidad;
        }
    }
}

Composición

class Motor {
    void encender() {
        System.out.println("Motor encendido");
    }
}

class Coche {
    Motor motor = new Motor();
}

Delegación

class Impresora {
    void imprimir() {
        System.out.println("Imprimiendo...");
    }
}

class Gerente {
    Impresora impresora = new Impresora();
    void imprimirReporte() {
        impresora.imprimir();
    }
}
